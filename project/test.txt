O Open ended questions can be whatever you want but the question must contain only one instance of a period or question mark, that is because the program identifies where the question is in the string using those symbols. All the text following the first instance of those punctuation characters will be the correct answer.
O How are questions separated? Questions are separated by line, each line represents a question.
